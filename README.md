## 📝 Descrição
Este repositório reúne códigos simples em HTML desenvolvidos durante as aulas de **DAW (Desenvolvimento de Aplicações Web)**. O objetivo é registrar a evolução no aprendizado de estruturação semântica e tags fundamentais da web.

## 📚 Conteúdo Praticado
- Estrutura básica de documentos HTML5.
- Formatação de textos, títulos (`h1`-`h6`) e parágrafos.
- Criação de listas ordenadas e não ordenadas.
- Inserção de links, imagens e tabelas simples.
- Introdução a formulários.

## 🛠️ Tecnologias Utilizadas
- **HTML5** (Semântica básica).

## 💻 Como Visualizar
1. Baixe ou clone o repositório.
2. Abra qualquer um dos arquivos `.html` diretamente no seu navegador preferido para ver o resultado da estrutura.
